Introduction

A multidimensional explorer of cause of death predictions. These cause of death predictions were generated by several algorithms computing verbal autopsy data of patient symptoms.

The parallel coordinates displays the symptoms, cause groupings and location colored by the specific algorithm used to compute the prediction.group.

Read this tutorial to learn more about parallel coordinates.

Controls

Brush: Drag vertically along an axis.
Remove Brush: Tap the axis background.
Reorder Axes: Drag a label horizontally.
Invert Axis: Tap an axis label.
Remove Axis: Drag axis label to the left edge.
